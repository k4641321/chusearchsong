# 中二查歌

一个使用beeware，toga框架写的查歌软件，歌曲数据来源于落雪咖啡屋

# 功能
- [x] 基本查歌
- [x] 分类查歌
- [x] 版本查歌
- [x] 定数筛选
- [x] 歌曲详情
- [x] 曲绘预览
- [ ] 歌曲试听
- [x] 收藏夹
- [ ] 别名查歌
